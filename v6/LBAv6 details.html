Pie Chart

Shows spending split by category for the current month. Hover or tap a slice to see the category name and exact amount. Updates live as you add expenses.

Upcoming Recurring Banner

Appears at the top when recurring expenses are due within 7 days. Each is shown as a colour-coded chip:

  • Red — due today or tomorrow
  • Amber — due in 2–3 days
  • Blue — due in 4–7 days

Month-End Forecast

Projects your total spend by end of month based on your current daily rate. Shows trend vs previous month (↑ higher / ↓ lower).

💡 Safe-to-Spend formula: (This month's income − fixed expenses − savings goal) ÷ remaining days in month. Set your savings goal in Settings.

💱 14. Multi-Currency Support
  • Every expense and income can be in any of 27 currencies
  • The app always stores the INR equivalent alongside the original amount
  • All totals, charts, and summaries display in your chosen base currency
  • When you select a non-base currency, the conversion rate is auto-filled from cached live rates
  • You can manually override the rate if needed
  • Exchange rates are cached for 6 hours — refresh manually in Settings or Exchange tab

🟢 16. Real-Time Sync

Changes made on one device appear on all other open devices within ~1 second — no page refresh needed.

Status Dot (top-right header)

Pulsing green — Live and connected. All changes sync instantly.
Spinning amber — Connecting to live sync.
Red — Sync error. Changes still save locally.
Gray — Offline.

What syncs in real time

Expenses · Income · Budgets · Goals · Goal contributions — all 5 tables sync live across all devices simultaneously.

🔔 17. Push Notifications

Daily reminders when a recurring expense is due within 3 days — even when the app is closed.

Enabling Notifications

1
Go to Settings → Recurring Reminders
2
Click 🔔 Enable
3
Allow the browser permission prompt
4
Status shows: "Active — you will receive daily reminders"

How Notifications Work

  • Fire at 07:00 AM IST every day
  • Show expense description, amount, and days until due
  • Tapping the notification opens the app directly to the Recurring tab
  • Each device subscribes separately — enable on phone and desktop independently

Browser Compatibility

BrowserStatusNotes
Chrome (PC + Android)✅ WorksOut of the box
Brave⚙️ Needs configEnable "Use Google services for push messaging" in brave://settings/privacy

📱 18. PWA — Install on Device

Install the app like a native app — no App Store required.

DeviceHow to install
Android (Chrome/Brave)Tap three-dot menu → Add to Home Screen or Install App
iPhone/iPad (Safari only)Tap Share button → Add to Home Screen · Requires iOS 16.4+ for push notifications
Windows/Mac (Chrome/Edge)Click the install icon in the address bar → Install

PWA Features

  • Works offline (service worker caches the app)
  • Auto-updates when a new version is deployed — no manual refresh needed
  • Full screen, no browser chrome
  • Push notifications (when enabled)
  • Installable icon on home screen / taskbar
🔀 19. Data Migration from V5

Bring all your history from the old single-file HTML app into V6.

Steps

1
Open the old V5 HTML file in a browser
2
Go to Settings → Export Data → Download JSON
3
In V6, go to Settings → Migrate from V5 → Choose V5 File
4
Select the JSON file — migration runs automatically
What migratesWhat does NOT migrate
All expenses (categories, tags, notes, recurring flags)Budgets — re-set in Budgets tab
All income recordsGoals — re-create in Goals tab
Original fingerprints (dedup protection)App settings (dark mode, base currency)
✅ Safe to run multiple times — duplicate records are detected by fingerprint and automatically skipped.
